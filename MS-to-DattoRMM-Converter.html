MS → Datto RMM Script Converter

Strip interactive prompts from Microsoft scripts for silent RMM deployment

Input — Microsoft Script
✓ Output — Datto RMM Ready
What this does: Microsoft's Defender onboarding/offboarding scripts are designed for interactive use — they include set /p prompts, pause commands, and timeout.exe calls that hang when run as a Datto RMM job. This tool strips those interactive elements while leaving all the actual onboarding logic untouched. The converted script exits with proper error codes so Datto can report job success/failure.
